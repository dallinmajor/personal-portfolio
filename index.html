Partial Twitter Clone Using jQuery

Tweet and other basic functionality

Below is a list of features that I have done and am still working on. This started out as an optional class project at DevMountain Bootcamp.

  • Initially, the Tweet button and the character count button are hidden (CSS).
  • When the user clicks on the textarea, the textarea doubles in size and the character count and Tweet buttons are revealed.
  • As the user types, the character count decreases.
  • When there are 10 or less characters, the character counter turns red.
  • If the user puts in more than 140 characters, the tweet button is disabled (and re-enabled when there are <= 140 chars).
  • When the user successfully inputs characters and clicks the “Tweet” button, a new tweet is created and added to the tweet stream in the main column, using the user’s fake profile image in the top left and username/fullname.
  • The tweet actions (Reply, Retweet, etc) only show up when you hover over that individual tweet. Otherwise, they are hidden.
  • The Retweets/timestamp/Reply areas are hidden by default. These only expand if you click on the tweet. I use jQuery animation to accomplish the reveal, similar to how it’s done on Twitter.com

    • Features that are not implemented yet and still to come:

  • Make timestamps similar to how they look on Twitter (1h, 18m, 1m) and use the jQuery "timeago" plugin to make them automatic.
  • Implement the icons for when a tweet is favorited/retweeted in the upper right of the tweet.
  • Clicking favorite to increase the favorite count.
  • Clicking retweet to increase the retweet count and take the text of the tweet and retweet it.
  • Implement the Bootstrap tooltips for when you hover over a user’s avatar image
  • Persist new tweets using local storage & persist new tweets using Firebase
  • Date: May 2017
  • Category: jQuery, CSS, HTML.
